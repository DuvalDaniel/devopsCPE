2-1 What are testcontainers?
Testcontainers est une bibliothèque Java permettant d'exécuter des conteneurs Docker légers pour les tests d'intégration. Elle est particulièrement utile pour tester des bases de données, des systèmes de messagerie ou d'autres dépendances externes sans nécessiter d'environnement de test complexe.
